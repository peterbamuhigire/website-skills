# Healthcare Provider Sector Brief

Copy this template to `docs/sector-brief.md` in your client project and answer each section.

**Reference complete example**: See `tour-travel/sector-brief-template.md`

## Brand Identity
- Type of practice: (Hospital, clinic, dental, specialist office?)
- Primary patient demographic?
- Key differentiator vs other providers?

## Visual Identity
- Visual personality: Clinical/formal? Warm/reassuring? Modern/tech-forward?
- Primary color palette (suggest: calming blues/greens + professional neutral)
- Accessibility first? (Large text, high contrast, keyboard navigation)

## Trust & Conversion
- Primary CTA: (Book appointment, Request info, Find doctor)
- Key credentials: (Board certifications, specialties, experience)
- Insurance accepted? Patient testimonials?
