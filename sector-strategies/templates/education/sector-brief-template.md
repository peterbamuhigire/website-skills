# School/Institution Sector Brief: Education

Copy this template to `docs/sector-brief.md` in your client project and answer each section.

See the **tour-travel sector** for a complete example template.

## Brand Identity
- What is the institution's core mission?
- Age group/student demographic served?
- Primary value proposition vs competitors?

## Visual Identity
- Visual personality: Academic/formal? Modern/progressive? Warm/welcoming?
- Primary color palette (suggest: professional primary + one accent)
- Typography: Serif or sans-serif for headers?

## Trust & Conversion
- What is the primary CTA? (Enroll, Apply, Request tour, Download prospectus)
- Key credentials or accreditations?
- Notable alumni or student success stories?
- Parent vs student messaging: Which audience first?

## Content & Messaging
- Top 3 academic programs/courses to highlight?
- Admissions requirements or process?
- Competitor websites for reference? (up to 5)

## Brand Colors

Define your primary brand colors as hex codes. These override sector defaults and guide all design decisions.

**Primary Brand Color**: (e.g., #1E40AF Primary Blue)
**Secondary Color**: (e.g., #059669 Healthcare Green)
**Accent Color**: (e.g., #D97706 Gold)
**Background Color**: (e.g., #F9FAFB Light Gray)
**Text Color**: (e.g., #1F2937 Neutral Dark)
